使用Python获取股票数据并可视化¶

通过Python可以使用pandas_datareader包通过雅虎财经获取股票数据，尝试了该方法但返回错误，经查询雅虎财经对API做了调整，pandas_datareader官网显示所支持的API列表已经没有雅虎财经了。

透过科学上网工具，找到了另外一个获取雅虎财经数据的功能包 -- yfinance。利用该功能包，可以使用相似于pandas_datareader的方法获取股票数据。

下面通过yfinance提取中美互联网公司的股票数据，并做可视化分析。

提取数据¶

考虑到公司的对比性，选取了中国互联网三巨头（BAT）阿里（股票代码：BABA），腾讯（0700.HK），百度（BIDU），以及美国互联网的三家巨无霸(FAG)：脸书（FB），亚马逊（AMZN），谷歌（GOOGL）。

获取数据需要设置起始日期和结束日期，我们要比较2018年以来这六家公司的股票走势，所以起始日期设置为‘2018-01-01’，结束日期为2019-08-27。

In [3]:
# 设置要提取的股票代码为中国的BAT和美国的谷歌、脸书、亚马逊
tickers = ['0700.HK', 'BABA', 'BIDU', 'GOOGL', 'FB', 'AMZN']

# 设定起始日期和结束日期
start_date = '2018-01-01'
end_date = '2019-08-27'

# 提取雅虎财经数据
data = yf.download(tickers = tickers, start = start_date, end = end_date)

可以看到，数据的行索引是交易日日期，列索引有两层，第一层是当天股票的开盘价，最高价等变量，第二层列标题是股票代码。

由于我们需要比较的是2018年以来的股价走势，所以只需考虑收盘价。数据中有两种收盘，Close和Adj Close，分别代表不复权收盘价和前复权收盘价。股票分红、转股等情况下股价会除权，因此计前复权收盘价更能体现股价长期的真实变化（收益）。关于复权的概念，可以参考这篇文章。

因此我们需要的数据在Adj Close列。

根据提取的数据绘制出了这6家公司股价的走势，每家公司的每股价格差距较大，亚马逊和谷歌每股股价高于1000美金，其他几家股价数值低于500，并且腾讯的股价单位是港币。从上图直观上可以看到2018年以来亚马逊的股价是有明显上涨的，其他几家公司的股价走势要么持平于2018年初，要么走低。

为了比较各公司股价的涨跌幅，需要对数据进行处理，将该股所有的股价都除以该股起始日期（本文中的2018年1月1日）的股价，即将股票绝对值变成相对值，这样就能比较2018年以来这六家公司的股票收益。我们设定100为起始日期基准价，之后股价的波动都是相对100这个基准价而言的，这样也能忽略掉不同货币单位的影响。

另外，由于腾讯是在港股上市，港股和美股的休市日期不同，因此股价曲线会有几处中断，代表该日是休市日，该股无交易。对于休市日股价，可以认为休市日收盘价等于上一个交易日的收盘价，这样就可以使用df.fillna()通过前值填充休市日的股价，使股价曲线无间断。

In [7]:
# 复制df的数据到df2
df2 = df.copy()

# 填充休市日股价为上一个交易日收盘价
df2.fillna(method='ffill', inplace=True)

# 使用for循环将每个公司的股价基于起始日收盘价折算，tickers为之前定义的公司股票代码组成的列表
# tickers = ['0700.HK', 'BABA', 'BIDU', 'GOOGL', 'FB', 'AMZN']
for ticker in tickers:
    
    # 该股全部收盘价除以起始日期收盘价，然后乘以100，股价按惯例保留2位小数
    df2[ticker] = round(df2[ticker] / df2[ticker].iloc[0] * 100, 2)

新的图表更清晰地显示出了6家中美互联网公司2018年以来的股票涨跌走势。6家公司中只有亚马逊和谷歌的股票实现了正收益率，其余4家均是负收益率。3家美国公司股价总体表现大大好于3家中国公司，BAT均录得-10%以上的负收益。

此外结合贸易战的时间点，可以看到贸易战的几个重大时间点都影响了6家公司的股价，尤其是2018年3月贸易战开战、2019年8-9月新一轮关税加征和2019年5月特朗普推特宣布额外5000亿美元商品加征关税都重创了6家的股价。在贸易战有所缓和的2018年12月至2019年4月，除百度外都实现了不小的涨幅。

  1. 亚马逊：亚马逊在2018年是华尔街的宠儿，股价在2018年初随着利润指标改善即开始大幅增长，除受2018年3月贸易战开战影响到导致股价调整外，不断扩大涨幅，至2018年9月达到了70%的收益，成为历史第二支万亿美元市值公司，创始人贝索斯也因此荣登全球首富。此后随美股股指起落，自身增速放缓，收益率至今降到了49%。不过股神巴菲特今年都开始买入了，本就是基金重仓的亚马逊继续下跌也难。

  2. 谷歌：过于依赖广告收入的谷歌股价走势相对稳定，波幅在20%左右，在大部分的时间里维持在正收益率，不过广告业务的下滑、流量获取成本的上升，广告点击率所带来的收入下降、欧盟不断的反垄断罚款导致公司利润受影响，今年4月财报不及预期后一个月内跌去20%，好在7月的最新财报超预期，保住了一年多来9%的收益率。谷歌未来的股价仍然会依赖广告业务的表现。

  3. 脸书：2018年的脸书很失意，几次服务器宕机事件甚至都引起股价下跌，能维持-1%的收益率靠的是2019年的强势财报。2018年3月受总统选举数据泄露事件影响，收益率排在垫底位置，此后虽一路上涨，但是2018年半年报不及市场预期，公布当天暴跌近20%，不幸成为史上第一支单日市值下挫超千亿美元的个股，财报显示脸书的用户增长和营收增速放缓，股价也一直萎靡至2019年1月。此后的财报数据好转，至今股价勉强收复失地，好在Instagram用户快速增长，这款App有机会助推脸书的股价。

  4. 阿里：阿里的财报相对稳健，营收增速还能维持在40%以上，尤其核心的电商业务依然在狂奔，下沉市场挖掘战略显成效，月活用户仍能保持15%以上的增长。图上可以看到，在今年5月前阿里的股价走势和脸书在大部分时间里很接近（阿里波幅略小），今年5月特朗普发推加剧贸易战后，阿里跌幅超过脸书，2018年至今收益率为-10%，已经是BAT三家中表现最好的。在实体经济下滑、国家刺激消费扩大内需的情况下，电商业务为主的阿里应该还能保持不错的增速，有助于推动股价的增长。

  5. 腾讯：2018年12月前曾是6家里表现股价最差的公司，不仅受贸易战影响，还被大股东MIH减持，自身利润增速放慢，行业版号审批问题导致游戏业务大受影响，引发市场担忧，2019年更被港股整体市场拖累，最近的8月受恒生指数大跌牵连以及一份市场不认可的财报，股价跌去10%，最终录得-22%的收益率。亮点是金融科技和企业服务收入大增，最近两个财报季增速都超过44%。另一好消息是最近几个月获得版号的游戏增多，游戏业务开始回暖。下一财报季的表现值得期待。

  6. 百度：相信不仅是这6家中最让人失望的公司，也是有代表性的中概股中最让人失望的公司，-57%的收益率甚至差过年轻人的第一支腰斩股小米。主营的搜索业务还在不断被人诟病。2018年5月前百度曾经回光返照创造股价新高，5月陆奇离开，股价也跟着其他中概股一起一路下跌，但百度没有反弹。该来的总会来的，百度在2019年5月首次发布亏损的财报，期间至今又跌去40%，市值相继被美团、京东、拼多多超过。旧业务增长乏力、AI业务又无法在短时间内带来利润，中概股中有太多比百度更值得投资的标的。

综上来看，2018年来依赖广告业务的公司股价相对低迷，这与全球经济增长放缓有关，互联网红利的消退也导致互联网行业月活用户增长慢。贸易战也一直是笼罩在全球经济之上的乌云，特朗普时不时就发推打个雷。消费概念的亚马逊和阿里分别是美中互联网巨头里表现最好的股票，过去一年里A股表现最好的也是消费股（白酒、酱油、家电、机场等）。所以，仓位里配置一部分消费股对于散户来说是个不错的选择。当然，除百度外，上文的另外5家公司仍然是行业里的翘楚，用户粘性也高，在可预见的3-5年内也都是值得投资的标的。

以上是本人对于6家公司股价走势的一些分析和拙见，我不是行业内的人，难免分析不足不够准确，还请见谅。
